My name is Conner Smith


Hello! I am an indie game developer from the United Kingdom, and I have been programming and experimenting with creating games since I was 12. I have always taken an interest in the creation process of many iconic mechanics from beloved games such as placeable portals from "Portal" and grappling hooks from the "Just Cause" series which both help to make their repectible franchises memorable and fun.
Here you can see some of the projects that I have had the opportunity of contributing to, and some of the personal projects and experiments I have been working on. I have a strong perseverance for learning new things and taking risks as well as tackling challenges, in order to make sure that I can grow as an individual as well as a creator overall. I have always been immensely curious in the technical side of games, and this is the drive for my future ambitions and projects.

College Projects

Very Hungry Catterpillar (Book themed College Project)

This is a project that I completed within 4 weeks based around a book called "The Very Hungry Catterpillar". This was the first Unity game that I made using self made art assets as well as original code that included basic movement, pause menu, and collection of items such as collectibles.
Highlights:
• First game in Unity and C#
• 100% Self made assets
• Includes first attempt at storing scores

Robin Hood's Cash Stash ("Myths and Legends" themed College Project)(Final Major Project Year 1)

This is a project that I spent 12 weeks on as my final major project for my first year at College. This was the first 3D game that I made with the use of imported 3D models as well as a fully explorable world with enemies that react to the player's behaviour.
Highlights:
• First 3D game in Unity
• Fully coded the player physics
• Sophisticated enemy AI
• Advanced UI (minimap)

Rapture Run (Culture themed College Project)

This is a project that I completed in 4 weeks. This is the first Unity game that I made which included accessibility options such as difficulty settings and a colour-blind friendly colour pallet as well as my first attempt at a randomised, infinite level with randomly spawning obstacles. Moreover, I fully created a background art piece which was completely tileable.
Highlights:
• Accessibility focused gameplay
• Procedural level generation
• Self-painted art assets

Coin Campo ("One More Go" themed College Project)

This is a project that I completed within 6 weeks based around the theme of wanting "One More Go". This is the first game I made that experimented with multiple pathways within a 2D space as well as different obstacles such as the key/door system which allowed for the player's pacing and advancements to be controlled.

Planet Zoo (October 2019 - October 2019 (1 day)):
As a side project, a select few people from my course (me included), were given the opportunity to visit Frontier Developments to take part in last minute QA testing before the game's release. We were given the task of playing through the tutorials and introduction levels and document any bugs or issues that we experienced.
